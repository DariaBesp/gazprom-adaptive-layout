# Gazprom Adaptive Layout 
Адаптивная верстка страницы Газпромбанк

## 🌐 Демо

[https://dariabesp.github.io/briaton-js/](https://dariabesp.github.io/gazprom-adaptive-layout/)

## 🚀 Запуск проекта

1. Клонировать репозиторий:
```bash
git clone https://github.com/DariaBesp/gazprom-adaptive-layout.git
```
2. Перейти в папку проекта
```bash
cd gazprom-adaptive-layout
```

3. Запустить локально:
* Вариант 1 (через VS Code с установленным расширением Live Server):
нажмите правой кнопкой на index.html → Open with Live Server

или
* Вариант 2 (через Node.js):
```bash
npx serve
```
